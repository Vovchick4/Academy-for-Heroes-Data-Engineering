# 🐍 Навчальний курс: Python від Основ до ООП

Цей курс розроблений для системного освоєння мови програмування Python. Він поєднує теоретичну базу, відео-лекції та практичні завдання для закріплення навичок.

## 📚 Структура навчання

### 1. [GitHub] Практика та Матеріали

- **[Module 1: Основи Python](https://github.com/Vovchick4/Full-Stack-Python-Development/tree/main/Module1)**
- Встановлення середовища.
- Змінні, типи даних та оператори.
- Робота зі списками, словниками та кортежами.
- Функції та модульність коду.
- **Об'єктно-орієнтоване програмування (ООП):** класи, об'єкти, спадкування та інкапсуляція.

### 2. [YouTube] Відео-ресурси

Для кращого розуміння тем рекомендується використовувати наступні плейлисти:

- **[PyNeng UK (Python для мережевих інженерів)](https://www.youtube.com/@pynenguk/playlists):**
- Це один із найкращих ресурсів для тих, хто хоче бачити реальне застосування Python.
- Глибокий розбір автоматизації, роботи з файлами та структурами даних.

- **[Плейлист: Основи Python](https://www.youtube.com/playlist?list=PL7vq4D0vOpQb-nious5dTzUF9GEMp6jiL):**
- Покрокові уроки, що ідеально доповнюють практичні завдання з модуля 1.
