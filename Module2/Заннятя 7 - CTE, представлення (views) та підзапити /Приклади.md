# 📘 1. SUBQUERY (ПІДЗАПИТ)

## 📊 Таблиці

### users

| id | name |
| -- | ---- |
| 1  | Ivan |
| 2  | Oleg |
| 3  | Anna |

---

### orders

| id | user_id | amount |
| -- | ------- | ------ |
| 1  | 1       | 500    |
| 2  | 1       | 1500   |
| 3  | 2       | 200    |

---

## 🧠 SQL

```sql
SELECT * 
FROM users
WHERE id IN (
    SELECT user_id FROM orders
);
```

---

## ✅ Результат

| id | name |
| -- | ---- |
| 1  | Ivan |
| 2  | Oleg |

---

## 💡 Пояснення:

👉 `Anna` не попала, бо немає замовлень

---

# 📘 2. EXISTS (краще ніж IN)

---

## 🧠 SQL

```sql
SELECT *
FROM users u
WHERE EXISTS (
    SELECT 1
    FROM orders o
    WHERE o.user_id = u.id
    AND o.amount > 1000
);
```

---

## ✅ Результат

| id | name |
| -- | ---- |
| 1  | Ivan |

---

## 💡 Пояснення:

👉 тільки `Ivan`, бо має замовлення > 1000

---

# 📘 3. CTE (WITH)

---

## 🧠 SQL

```sql
WITH big_orders AS (
    SELECT user_id, amount
    FROM orders
    WHERE amount > 1000
)
SELECT *
FROM big_orders;
```

---

## ✅ Результат

| user_id | amount |
| ------- | ------ |
| 1       | 1500   |

---

## 💡 Пояснення:

👉 створили “тимчасову таблицю”

---

# 📘 4. КІЛЬКА CTE

---

## 🧠 SQL

```sql
WITH users_total AS (
    SELECT user_id, SUM(amount) as total
    FROM orders
    GROUP BY user_id
),
rich_users AS (
    SELECT user_id
    FROM users_total
    WHERE total > 1000
)
SELECT *
FROM users
WHERE id IN (SELECT user_id FROM rich_users);
```

---

## ✅ Результат

| id | name |
| -- | ---- |
| 1  | Ivan |

---

## 💡 Пояснення:

👉 тільки Ivan витратив > 1000

---

# 📘 5. VIEW

---

## 🧠 SQL (створення)

```sql
CREATE VIEW user_totals AS
SELECT user_id, SUM(amount) as total
FROM orders
GROUP BY user_id;
```

---

## 🧠 Запит

```sql
SELECT *
FROM user_totals
WHERE total > 1000;
```

---

## ✅ Результат

| user_id | total |
| ------- | ----- |
| 1       | 2000  |

---

## 💡 Пояснення:

👉 View = збережений запит

---

# 📘 6. РЕАЛЬНА ЗАДАЧА

## 🧠 Умова:

* витрати > 1000
* більше 1 замовлення

---

## 🧠 SQL

```sql
WITH stats AS (
    SELECT user_id,
           SUM(amount) as total,
           COUNT(*) as orders_count
    FROM orders
    GROUP BY user_id
)
SELECT *
FROM stats
WHERE total > 1000
AND orders_count > 1;
```

---

## ✅ Результат

| user_id | total | orders_count |
| ------- | ----- | ------------ |
| 1       | 2000  | 2            |

# 📘 7. RECURSIVE CTE

---

## 📊 Таблиця categories

| id | parent_id | name  |
| -- | --------- | ----- |
| 1  | NULL      | Root  |
| 2  | 1         | Child |
| 3  | 2         | Sub   |

---

## 🧠 SQL

```sql
WITH RECURSIVE category_tree AS (
    SELECT id, name, parent_id
    FROM categories
    WHERE parent_id IS NULL

    UNION ALL

    SELECT c.id, c.name, c.parent_id
    FROM categories c
    JOIN category_tree ct ON c.parent_id = ct.id
)
SELECT * FROM category_tree;
```

---

## ✅ Результат

| id | name  | parent_id |
| -- | ----- | --------- |
| 1  | Root  | NULL      |
| 2  | Child | 1         |
| 3  | Sub   | 2         |

---

# 🎯 ВИСНОВОК (дуже важливо)

👉 Коли що використовувати:

* **Subquery** → швидко і просто
* **EXISTS** → великі дані
* **CTE** → складна логіка
* **VIEW** → повторне використання
* **RECURSIVE CTE** → дерево
