# 📘 🧩 Домашка 1 — SQL Function

## 📊 Таблиця `products`

```sql
CREATE TABLE products (
    id INT,
    name VARCHAR(50),
    price DECIMAL
);
```

### Дані:

| id | name  | price |
| -- | ----- | ----- |
| 1  | Phone | 1200  |
| 2  | TV    | 800   |
| 3  | Book  | 200   |

---

## ❓ Завдання

Створи функцію:

### 👉 `get_final_price(price)`

Логіка:

* якщо `price > 1000` → знижка 15%
* якщо `price BETWEEN 500 AND 1000` → знижка 10%
* якщо `< 500` → без знижки

---

## ✅ Очікуване використання

```sql
SELECT name, get_final_price(price) AS final_price
FROM products;
```

---

## 💡 Що перевіряється

* `CASE WHEN`
* правильний `RETURNS`
* відсутність side effects

# 📘 🧩 Домашка 2 — Stored Procedure

## 📊 Таблиця `users`

```sql
CREATE TABLE users (
    id INT,
    name VARCHAR(50),
    balance DECIMAL
);
```

---

## ❓ Завдання

Створи процедуру:

### 👉 `withdraw_money(user_id, amount)`

Логіка:

1. Перевірити баланс користувача
2. Якщо `balance < amount` → викинути помилку
3. Якщо достатньо → списати гроші

---

## ✅ Очікуваний виклик

```sql
CALL withdraw_money(1, 200);
```

---

## 💡 Що перевіряється

* `IF`
* `SELECT INTO`
* базова бізнес-логіка
* side effects (UPDATE)

---
