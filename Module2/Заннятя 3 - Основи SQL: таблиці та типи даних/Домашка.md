# 🏠 Домашнє завдання

## 🎯 Тема: Реалізація структури БД через SQL

---

## 📌 Контекст

На попередньому занятті ти:

* обрав тему
* описав як з’являються дані
* визначив OLTP / OLAP
* розписав роль Data Engineer

Тепер потрібно:

👉 **реалізувати структуру таблиць через SQL**
👉 побудувати реальну модель даних

---

# 🔹 Завдання 1 — Проєктування структури

На основі свого кейсу створити мінімум:

* 4 таблиці
* мінімум 1 зв’язок 1:M
* якщо логіка дозволяє — 1 зв’язок M:N

### Обов’язково:

* PRIMARY KEY
* FOREIGN KEY
* NOT NULL
* хоча б один UNIQUE
* хоча б один CHECK
* хоча б один DEFAULT

---

# 🔹 Завдання 2 — Реалізація SQL

Написати:

### 1️⃣ CREATE TABLE для всіх таблиць

З правильними типами даних:

* INT
* DECIMAL (для грошей)
* VARCHAR
* BOOLEAN
* TIMESTAMP

---

### 2️⃣ INSERT

Додати:

* мінімум 3 записи в головну таблицю
* мінімум 5–10 записів сумарно

---

# 🔹 Завдання 3 — Перевірка даних

Написати запити:

1. SELECT *
2. SELECT DISTINCT
3. SELECT з WHERE
4. SELECT з AND / OR
5. SELECT з BETWEEN або IN

---

# 🔹 Завдання 4 — ERD

Побудувати ERD на основі написаних таблиць.

Показати:

* PK
* FK
* тип зв’язків
* якщо є M:N — показати проміжну таблицю
